setInitCueStart()method 
public function setInitCueStart():void

This will determine what the initial state should be and also construct a CuePointSegment object to set what portion of the inital video should play.

The initCueSegment Object is composed of:

  • name: the video.cueArray[n].name String from the video MetaData
  • start: the video.cueArray[n].time Number from the video MetaData
  • end: the video.cueArray[n+1].time Number from the NEXT cue point which serves as the current segment end point

NOTE:setting an initial start value that exceeds the amount of video that has been loaded is not advised
